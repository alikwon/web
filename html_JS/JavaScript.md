# 자바 스크립트

## 1. 자바스크립트의 활용

- 인터넷 연결 없이 웹 브라우저에서 실행가능한 웹어플리케이션
  - computer 브라우저(chrome)에서 실행됨

- 데이터의 출력
  - alert() 함수 : 문자열, 숫자, 함수, 객체, 배열(객체)
  - console.log() : 문자열, 숫자, 함수, 객체, 배열(객체)
- 데이터 : 문자
  - 문자 표현 ==> ' ' 또는 " "
- 데이터 : 숫자
  - 정수와 실수를 구분하지 않음
- 연산자 : 산술연산자 → + - * / %
  - 우선순위는 () 가 최우선 , 기본순서와 동일
- 데이터 : bool true / false
- 비교연산자 : 결과는 true / false
- 변수 : var 변수이름;
